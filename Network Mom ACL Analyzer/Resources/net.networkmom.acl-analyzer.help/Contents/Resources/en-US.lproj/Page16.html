Cisco Network Object-Group Configuration

Cisco object-groups are inconsistent between platforms. Here are notes from my research on different platforms, including the links to the documentation where I found each object-group specified.

What is disturbing is that the IOS platforms use do-not-care-bit configurations for ACLs, but netmask configuration for object-group networks. Meanwhile NX-OS allows /prefix format (good!) but also allows do-not-care-bit configuration for object groups.
