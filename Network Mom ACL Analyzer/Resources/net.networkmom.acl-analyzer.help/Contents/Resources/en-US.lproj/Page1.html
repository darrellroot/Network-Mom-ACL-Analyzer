IOS-XR Limitations

In addition to the general limitations, Network Mom ACL Analyzer has many limitations regarding IOS-XR ACLS. Here are _some_ of those limitations:

  • Some ACL features (including even the ability to apply a L3 ACL to an interface) are IOS-XR version, platform, or linecard-specific. The ACL analyzer does not know what type of box you have and cannot report related errors.

  • Nested object groups on some IOS-XR platforms can only be 4 layers deep. The ACL analyzer does not report if it finds object groups nested greater than 4 deep.

  • ACL-Chaining / Multi-ACL / Common-ACL configurations are not supported.

  • ACL-based forwarding is not supported.

  • Receive access control lists are not supported

  • IPv6 wildcard masks are not supported. Use CIDR notation. “You’ll thank me later.” - Adrian Monk
