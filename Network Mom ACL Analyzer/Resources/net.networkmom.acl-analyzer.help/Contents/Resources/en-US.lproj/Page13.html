Release Notes

Version 1.6 July 26 2019

Fixed font-related bug

Version 1.5 July 25 2019

New Feature: Duplicate ACE detection.

New Feature: Random ACL generation for testing.

New Feature: Allows optional sequence numbers in IOS ACLs.

Improved error reporting for misaligned ip/prefix and do-not-care-bit situations.

Multiple bug fixes.

Changed default-font to fixed-width.

Version 1.4 July 22 2019

New feature: IOS-XR IPv4 ACL analysis including object-groups.

Removed feature: ASA object-group service configurations are currently disabled. Earlier versions had an error in logic which could have resulted in incorrect permit/deny calculations. ASA object-group network configurations remain active.

IOS and ASA ACL analysis were completely reimplemented, resolving several bugs.

Version 1.3 July 17 2019

Bug fix only release. Significantly improved parsing of IOS, ASA, and NX-OS IPv4 ACLs.

Version 1.2 July 15 2019

Added support for NX-OS IPv4 ACLs, including object-groups.

Version 1.1 July 12 2019

Dramatically improved performance dealing with large access-lists. This version successfully analyzed a 50,000-line Cisco IOS IPv4 ACL in under 15 seconds.

Version 1.0 July 8 2019

Initial public release with support for IOS and ASA IPv4 ACLs.
